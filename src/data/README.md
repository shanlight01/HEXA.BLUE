# 📁 `src/data`

Contient les données factices (mock data) utilisées pour que l'application soit fonctionnelle immédiatement, même sans connexion à la vraie base de données.

## Contenu
- `mockProviders.ts` : Une liste de 10 prestataires togolais fictifs. Contient leurs prénoms, compétences, et surtout leurs **coordonnées GPS** basées sur des points de repère réels de divers quartiers de Lomé (Agoè, Adidogomé, Deckon, etc).
