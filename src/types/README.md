# 📁 `src/types`

Ce dossier contient les définitions Typescript (les "contrats" de données).

## Contenu
- `index.ts` : Définit la forme exacte de chaque objet dans le code (Ex: Un `User` doit avoir un `uid`, un profil de service a besoin d'un tableau `skills`, etc).
- Assure qu'aucun bug de typage ne survienne en obligeant les développeurs à respecter ces structures.
