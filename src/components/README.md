# 📁 `src/components`

Ce dossier contient tous les composants d'interface utilisateur (UI) réutilisables.

## Structure
- `/ui` : Les petits composants indépendants "briques"
  - `SearchBar.tsx` : La barre de recherche principale
  - `ProviderCard.tsx` : La carte détaillée d'un prestataire (avec étoiles, etc.)
  - `WhatsAppButton.tsx` : Le bouton de contact avec lien dynamique
  - `CategoryPill.tsx` : Les boutons ovales pour filtrer (Ex: "Plomberie")
- `/layout` : Les gros composants de structure
  - `Header.tsx` : La barre de navigation en haut de l'écran (sur grand écran)
  - `MobileNav.tsx` : Le menu de navigation collé en bas de l'écran (sur téléphone)
